# Code of Conduct

This project follows the **Contributor Covenant version 2.1**. The canonical
text is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — as a contributor, maintainer, issue
reporter, or any other role — you agree to abide by that code in all
project-related spaces (issues, pull requests, discussions, chat, events,
and any one-to-one communication that touches the project).

## In short

- Treat people with respect. Disagreement is fine; personal attacks aren't.
- Assume good intent in others' messages; ask for clarification before
  reacting.
- Keep technical discussion technical. Politics, religion, and other
  identity-based debate belong elsewhere.
- The maintainers have the final call on what's appropriate in
  project-related spaces, and may remove or restrict participation from
  anyone who repeatedly disrupts that.

## Enforcement

Report incidents privately to **conduct@cosmicflow.space**. Reports are
handled by the maintainer team. We aim to acknowledge within 72 hours and
respond with a decision within 14 days for most cases.

Enforcement actions follow the Contributor Covenant guidance: private
reminder → public warning → temporary ban → permanent ban, scaled to the
severity and pattern of the behavior.

## Scope

This Code of Conduct applies wherever the project operates — including, but
not limited to: GitHub repositories under the `cosmicflow-space` org,
release announcements, and any official social-media accounts representing
the project.
