# Telegram guide screenshots (optional)

Drop screenshots here to illustrate the steps on the in-app guide page
(`/telegram-guide`). Each step shows its image **only if the file exists** —
missing files render nothing, so the guide always works without them.

Filenames the guide looks for:

| File           | Step it illustrates                          |
|----------------|----------------------------------------------|
| `step-1.png`   | Installing Telegram                          |
| `step-2.png`   | Finding @BotFather                           |
| `step-3.png`   | `/newbot` → name + username → token         |
| `step-4.png`   | Pasting the token into Mnemos → Telegram    |
| `step-5.png`   | Enable + Generate pairing code              |
| `step-6.png`   | Searching/opening your bot → Start          |
| `step-7.png`   | Sending `/pair <code>` → ✅ Paired           |

Use your **own** screenshots (license-clean). **Redact any secrets** —
especially the bot token in step 3 — before adding them. Keep them reasonably
small (phone screenshots ~1080px wide are fine).
